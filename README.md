# CourseWorkTRPO
Курсовая работа по ТРПО

После клонирования репозитория для начала следует:
#1) Зайти в папку backend\WebApi и открыть там файл WebApi.sln (в любой IDE, например VS Code, Visual Studio, Rider)

#2) Перед компиляцией проекта выберите один из способов
	# Проверьте файл 'appsettings.json'
	Убедитесь, что строка подключения соответсвует вашей (базу данных создавать не нужно, если вы используете 2 способ)
	
	# Способ 1
	Зайдите в папку backend\WebApi\Backups восстановите самую свежую базу данных
	
	# Способ 2
	Установите инструмент dotnet ef (если не установлен)
	dotnet tool install --global dotnet-ef
	
	Примените миграции
	dotnet ef database update
#3) Откройте папку 'frontend/client-app/' в IDE, например VS Code
	В терминале VS Code пропишите команду 'npm install' для установки всех пакетов.
	
#4) Сначала запускайте сервер (скомпилируйте проект или запустите .exe файл проекта)

#5) Запустите сайт клиента
	Применив команду 'npm start' в VS Code
